O que é a LGPD?

A Lei Geral de Proteção de Dados (LGPD) é uma legislação brasileira que regula a coleta, armazenamento e uso de dados pessoais, visando garantir mais segurança e transparência para os cidadãos.

Princípios da LGPD

A lei é fundamentada em alguns princípios essenciais, como:

  • Finalidade: os dados devem ser coletados para propósitos legítimos e específicos.
  • Adequação: o uso dos dados deve ser compatível com a finalidade informada.
  • Necessidade: apenas os dados indispensáveis devem ser coletados.
  • Transparência: os titulares devem ser informados sobre como seus dados serão utilizados.
  • Segurança: medidas de proteção devem ser adotadas contra acessos não autorizados.

Direitos dos Usuários

A LGPD garante diversos direitos aos titulares dos dados, tais como:

  • Confirmação da existência do tratamento dos seus dados.
  • Acesso aos dados coletados.
  • Correção de informações incorretas.
  • Solicitação de exclusão de dados.
  • Revogação do consentimento de uso.
